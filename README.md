## :computer: Get Started:

* Clone this repository:
```
git clone https://github.com/claudiaglez/LeapYear.git
```

* Navigate to the directory:
```
cd LeapYear
```

* Install dependencies:
```
npm install
```

## :test_tube: Test Execution

* Be sure to be in the right directory:
```
cd LeapYear
```

* Run the test:
```
npm test
```
